# Reviewer ORCID  (essential 🔴)

### Description/Meaning

The [ORCID](https://orcid.org/) associated with the reviewer

### **🤖**Examples

#### Crossref

```
Contributor / reviewer / ORCID
```

#### DataCite

```
nameIdentifier / SchemeURI / nameIdentifierScheme
```

#### DocMaps
